## 二 · 物无非彼 · 物无非是 (三清互为彼此)

「物无非彼 · 物无非是 · 自彼则不见 · 自是则知之」(庄子·齐物论)

**解** — 一物从不同角度看 · 彼此互为 · 不立对立.

### 2.1 · SP 7 态三层同步 (一即三 · 三即一)

| 层 | 字串 | 备 |
|---|---|---|
| 三清 (SDK) | `messages[].role = "system"` | 用户层入 |
| 二清 (web) | `passthrough` / `dao` / `usernote` / `prepend` / `append` / `override` / `custom` | UI 显字 |
| 一清 (dao_proxy) | `bypass` / `dao` / `usernote` / `prepend` / `append` / `override` / `custom` | 字串对齐 (passthrough = bypass alias) |

**真转链** (二清 web → 一清 dao_proxy):

```js
// web/dao_app.js:1541 (印 123 之实)
const strategy = mode === "passthrough" ? "bypass" : mode;
fetch(VM_URL + "/v1/system/prompt", {
  method: "POST",
  body: JSON.stringify({ strategy, customSp }),
});
```

**自彼则不见** — 从二清看不见一清之 `bypass` 字 (仅见 `passthrough`)
**自是则知之** — 但二清之 `passthrough` 真转后即一清之 `bypass` · 同物异名

### 2.2 · 三协议适配器 (一即三 · model 智能路由)

```js
// dao_proxy.js § 11 主路由 · 印 104 智能路由
POST /v1/chat/completions    → handleOpenAI
POST /v1/messages            → handleAnthropic
POST /v1beta/models/X:gen... → handleGemini

// model 含 windsurf|swe|cascade|sonnet|opus → windsurf 池
// 否则 → devin cloud wss
```

**三协议非分立** · 互转一致语义 · 三即一.

### 2.3 · 双反代 (windsurf + devin cloud · 一 VM 两轨)

| 反代 | 上游 | 协议 | 端点 | 真态 |
|---|---|---|---|---|
