```text
window.daoOAuth.getClientId()
  ↓ 综合调
_resolveClientId() · 高 → 低优先返
  ├─① _readUrlParam()      : URL ?dao_oauth_client_id=Ov23li...
  │                         (一次性 · 高优 · 分享/调试 · 不写盘)
  ├─② _readLocalStorage()  : localStorage 'dao_oauth_client_id'
  │                         (持久 · 一次为·万次用 · admin 面板填)

                              (placeholder · 主公未配 · isConfigured()=false)
```

## API 暴 (`window.daoOAuth.*`)

| 名 | 入 | 出 | 说 |
|----|----|----|----|
| `getClientId()` | () | string | 当前生效 client_id (4 源解析后) |
| `setClientId(v)` | (string) | string (=v) | 存入 localStorage · 立活 · 拒 placeholder/short/空 (throw) |
| `clearClientId()` | () | string (回退后值) | 清 localStorage · 回退 4 源链 (URL > window > DEFAULT) |
| `whichSource()` | () | "url_param" / "localStorage" / "window_global" / "default_placeholder" | 当前值来自哪一源 (admin 面板用) |
| `isConfigured()` | () | bool | client_id 是否真值 (排 placeholder) |
| `setupHint()` | () | { configured, client_id, url, hint, steps[] } | 主公建 OAuth App 之 6 步 hint |
