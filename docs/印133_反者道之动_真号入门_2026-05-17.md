### `POST /admin/wam/use`

**入**:
```json
{
  "index": 107,
  "mode": "auto"  // 'auto' | 'token-direct' | 'email-login'
}
```

**仅本机访** (帛书三十六): 远程客户 → 403 `wam_use_localhost_only`

**mode 辨**:

| mode | 路径 |
|------|------|
| `auto` (默) | item 有 token → token-direct; item 有 pw → email-login |
| `token-direct` | server 端读真 token → 直推 `WS_POOL_STATE.keys[]` (跳登) |
| `email-login` | server 端读真 email+pw → `_signin_orchestrate` (印 129 三步链) → 出 ws-* → 入池 |

**出** (token-direct):
```json
{
  "ok": true,
  "mode": "token-direct",
  "action": "pool.push",
  "apiKey": "auth1_bkkarc…",
  "tokenKind": "auth1",
  "email": "...",
  "count": 1
}
```

```json
{ "ok": true, "duplicate": true, "mode": "token-direct", "apiKey": "...", "count": 1 }
```
